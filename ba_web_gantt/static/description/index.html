Framework for add Gantt view

This is only framework. For use:

1. Add view for object

<ganttview start_date="gantt_start_date" stop_date="gantt_stop_date">

</ganttview>

where you specify the start and end fields.

2. Add in action to field "view_mode":

ganttview

Dynamic chart. It is possible to move terms and also to open tasks directly on the chart. Shows dependences between tasks and subtasks. Terms taking into account date and time.
